Exemplos De Formatação De Textos

Vamos ver como é feito cada um desses formatação.
Qual tag usar pra cada um.

  • Texto negrito - <b>Texto negrito</b>
  • Texto forte - <strong>Texto forte</strong>
  • Texto enfatizado - <em>Texto enfatizado</em>
  • Texto itálico - <i>Texto itálico</i>
  • Texto grifado - <mark>Texto grifado</mark>
  • Texto grande - <big>Texto grande</big>
  • Texto pequeno - <small>Texto pequeno</small>
  • Texto taxado - <del>Texto taxado</del>
  • Texto inserido - <ins>Texto inserido</ins>
  • Texto sublinhado - <u>Texto sublinhado</u>
  • esse é umTexto subscrito - <sub>Texto subscrito</sub>
